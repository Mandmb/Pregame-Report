# Pregame Report Website

Open `index.html`, upload the CSVs, select the opponent/logo, generate the report, and export PDF.

Updates in this version:
- SBA By Player PDF table now uses count columns SBA0-0 through SBA3-2 instead of SBA2/SBA3 groups.
- Header has Leones del Escogido home badge plus selectable opponent badge for Licey, Aguilas, Toros, Estrellas, or Gigantes.
- Header title remains editable.
